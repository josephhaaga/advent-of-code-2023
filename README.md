# advent-of-code-2023
Solutions to Advent of Code 2023

## Usage
To install...
```bash
$ virtualenv venv
$ . venv/bin/activate
$ python3 -m pip install -r requirements.txt
```

To run tests
```bash
$ python3 -m pytest
```

### TDD
While developing...
```bash
$ PYTHONPATH=. ptw --beforerun clear
```
